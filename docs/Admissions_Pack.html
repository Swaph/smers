Financial Aid Committee Impact Record (250 Words)

I served as the lead researcher, systems designer, and full-stack developer for the Smart Maternal Emergency Response System (SMERS), an applied academic project demonstrating how software can improve emergency maternal transport coordination in resource-constrained settings. My contribution covered the full research-to-product lifecycle: problem framing, literature-informed architecture, database and API design, geospatial dashboard implementation, simulation workflow design, and final technical documentation. I also authored and consolidated the concept papers, project proposal, progress reports, software design documentation, UML artifacts, and final thesis package.

From an intellectual impact perspective, the work translates public-health coordination challenges into a measurable engineering system with traceable operational metrics such as queue behavior, assignment logic, and mission closure records. From an entrepreneurial perspective, I positioned SMERS as a social-impact venture concept: a scalable dispatch and visibility platform that can be adapted for county-level emergency logistics, beginning with a locally grounded Laikipia-centered prototype. This establishes a credible foundation for pilot partnerships, product refinement, and future deployment pathways.

My role was not advisory; I built and integrated the core platform components end to end and published verifiable artifacts in a public repository. The evidence below includes formal project documents and source artifacts that confirm authorship, technical scope, and execution depth.
